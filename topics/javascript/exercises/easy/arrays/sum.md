Для вычислений можно использовать как циклы, так и `Array.reduce`.

Пример входных данных:
```js
const arr = [1, 2, 3];
```

Пример функции
```js
function sum(arr) {
    // <-- Добавь код для вычисления
}
```

Пример результата:
```js
6
```
